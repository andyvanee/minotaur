What is it?

Minotaur began as a minimalistic CSS framework. Since Bootstrap 3 came along and implemented the majority of these ideas, it's been converted to leverage Bootstrap.

Starting at < 20kb!!

Bootstrap alone clocks in at around 120kb. Minotaur only includes the core of Bootstrap, along with normalize.css and the handy mixins from LessHat. This gives you a lean and powerful starting point to implement your own sites.

Usage

Start by importing minotaur.less. Take a look there to see what's included. If you need other parts of bootstrap, just import what you need.
